# 债市周观察长图项目

## 项目目标

把“金葵花债市周度复盘”Word 素材，转成招商银行金葵花风格的债市周观察长图，并沉淀成可复用的长图生产项目。

当前素材周期：2026-05-15 至 2026-05-21。

## 当前状态

当前输出属于第一版自动化草稿，尚未验收，不建议作为最终发布版本使用。

主要原因：

- 首屏和部分卡片中间空白过多，版式没有根据内容高度重新收紧。
- “债市分析”板块存在文字与图标/背景块叠加的问题。
- 当前脚本以固定坐标覆盖旧图，缺少自动排版和碰撞检测。
- 原版 PSD 目前采用“原 PSD + 顶层新版长图可见层”的方式保存，方便查看和临时微调，但不是完全可编辑的新版文字/图表 PSD。

详细问题见 [docs/待解决.md](docs/待解决.md) 和 [docs/问题记录.md](docs/问题记录.md)。

新的基础长图方案见 [docs/superpowers/specs/2026-05-25-photoshop-basic-long-image-design.md](docs/superpowers/specs/2026-05-25-photoshop-basic-long-image-design.md)。

## 素材口径

- `金葵花债市周度复盘20260521.docx`：原始素材，用于理解结构和对比。
- `金葵花债市周度复盘20260521 - 修改版本.docx`：本轮主要内容来源。
- `债市周报.psd`：原始 Photoshop 模板。
- `长图/原版.png`、`长图/固收+.png`、`长图/债.png`：旧版参考长图。
- `长图/20260521新版/`：本轮生成结果。

三版 PNG 的差异主要在首屏开头文案；下方主体结构基本一致。策略部分按本轮要求沿用原策略，没有做实质改写。

## 现有脚本

- `scripts/build_long_images.py`：从 Word 提取文本、表格和图表数据，并基于旧长图底图生成三版 PNG。
- `scripts/list_psd_layers.jsx`：在 Photoshop 中导出 PSD 图层位置，用于理解模板结构。
- `scripts/make_original_psd_overlay.jsx`：在原 PSD 中贴入新版原版 PNG，另存为新版 PSD。
- `scripts/export_psd_right_preview.jsx`：用 Photoshop 从新版 PSD 导出右侧长图预览，用于校验 PSD 合成效果。

## 建议的下一阶段方向

下一阶段不要继续在固定坐标上补丁式覆盖，建议重做排版生成逻辑：

1. 先确定信息架构：首屏、债市表现、债市分析、后市展望、配置策略、风险提示。
2. 设计一个“内容块自适应高度”的长图布局系统，避免大空白。
3. 把图标、标题、正文、图表作为独立组件，增加安全边距和碰撞检测。
4. 对三版首段文案做统一字数约束，避免某一版撑开或留白。
5. 再决定 PSD 是否需要完全可编辑。如果需要，应重建 PSD 文本层和图表层，而不是只贴顶层成图。
